# weatherApp
날씨 앱
getCurrentPositionAsync()로 위치를 얻어낸 후
https://openweathermap.org/에서 제공하는 오픈 api를 이용해 얻어낸 위치에 00:00:00 시간의 날씨를 가져왔습니다.
날씨마다 아이콘을 다르게 적용해 만들었습니다.
